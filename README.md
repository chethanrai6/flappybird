# Flappy Wings

A modern, high-fidelity web implementation of **Flappy Wings** built with HTML5 Canvas, Vanilla CSS, and JavaScript (Vite).

## Features
- **Modern Dashboard UI**: Sidebar navigation, hero sub-panel, quick stats cards, and theme switcher (Light / Dark mode).
- **60 FPS Canvas Game Engine**: Smooth physics, bird tilt dynamics, motion trails, custom green vector pipes, and parallax background layers.
- **Game Over & Pause Screens**: Exact match to design mockups with score recap, crashed bird sprite with feather particles, and replay actions.
- **Audio Synthesizer**: Web Audio API retro-modern sound effects for flap, score point, crash, and button clicks.
- **Modals & Stats**: Leaderboard, How to Play guide, and Settings (Sound FX & Difficulty).

## Quick Start
```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
